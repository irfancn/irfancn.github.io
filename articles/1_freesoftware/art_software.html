Free softwares in Engineering

CAE and CAD tools has been instrumental in the latest advances, enabling engineers to model and optimize complex designs, reducing development costs and time while improving product performance and reliability. The ability to effectively utilize these software tools can lead to improved design, simulation, analysis, and optimization of engineering systems and processes. In general, proprietary software is used in industry and academia for ease of maintenance and better usability. Despite having these qualities, they may be out of reach for an individual or a small business, mainly due to the high cost. Apart from the financial aspect, using proprietary software can sometimes be a hassle. Some of my interactions with software manufacturer from my past are paraphrased below.

"Oh! you want to change the output format and process it further? that would require an additional toolset, costing an extra 30%".

"Sorry, the current version of the software cannot interact with a 3rd party package. But no worries, we have a software in our suit that can do little bit of that work".

In this article I would like to discuss commonly used software for various engineering applications and their free alternatives. While free software has its own share of problems, in certain aspects it can be superior to proprietary software. I hope that this will be useful to someone, either to find a free alternative of software, or to discover more about different engineering tools.

    2. The main objectives of this article are,

    2. Suggest free alternatives to some commonly used proprietary software.
    3. Promoting open source software and packages.
    4. To help me keep up to date with developments in engineering tools.

  2. Engineering tools and applications

    3. Engineers use software tools to design, simulate, analyse and optimise various engineering systems and processes. The software can be categorised according to the area of engineering application.

    2.1 Multiphysics modeling

    Most systems are simplified based on the physical phenomena needs to be observed, since it is much simpler to uncouple these phenomena and examine them separately rather than having a complex system where multiple phenomena are coupled. This way of systematically analysing a system simplifies the system to make the process simpler and faster. Some of the important physical phenomena for a system analysis are,

    1. Structural mechanics
    2. Thermal analysis
    3. Electromagnetics
    4. Fluid dynamics
    5. Acoustics

    2.2 Engineering tools & techniques

    There are many engineering techniques available to address the analysis in the above areas. Some of the commonly used software tools for engineering applications are,

    • Computer-Aided Design (CAD): CAD software is used to create 2D and 3D models of engineering components and systems. Engineers use CAD software to design and visualize components, assemblies, and systems before they are built.
    • Finite Element Analysis (FEA): FEA software is used to simulate the behavior of engineering systems under various loading conditions. Engineers use FEA software to analyze stresses, deformations, and other physical phenomena that occur in engineering systems.
    • Computational Fluid Dynamics (CFD): CFD software is used to simulate the behavior of fluids and gases in engineering systems. Engineers use CFD software to analyze fluid flow, heat transfer, and other physical phenomena that occur in engineering systems.
    • Simulation software: Simulation software is used to simulate the behavior of engineering systems under various conditions. Engineers use simulation software to predict the performance of engineering systems and optimize their design.
    • Project management software: Project management software is used to manage and track engineering projects. Engineers use project management software to keep track of project timelines, budgets, and milestones.
    • Data analysis software: Data analysis software is used to analyze and interpret data collected from engineering systems. Engineers use data analysis software to identify patterns, trends, and anomalies in data, and to make informed decisions based on the results.

    4. These are some of the aspects that are generally considered when choosing software. This comparison is general as there is a huge range of software available and many of the criteria are subjective.

    Remarks

    • It is possible to get student or free versions of most software. Although these often have limited functionality, they may be sufficient in some cases.
    • For beginners, it is better to stick with the proprietary software, especially for the documentation and tutorials. Also, some analyses may require pre- and post-processing, and free software may require additional packages. This can be inconvenient at the beginning.
    • Some of the free software requires Linux as an operating system, but WSL (Windows Subsystem for Linux) in Windows may help here. Similarly there are programs like 'Wine' to run Windows programs in Linux. Or another OS can be installed inside a virtual machine as well.
    • In between these to catogories, there is a grey area with cheaper CAE software. For many applications, this can also be a good choice.

  4. Propriatary software and Free alternatives

    5. From the vast collection of software for various engineering applications, only 3 influential areas are considered here: CAE, CAD and data analysis. The most commonly used software in these areas is listed and bundled according to its application and features.

    4.1 CAE

    Computer Aided Engineering (CAE) uses various tools such as FEA, CFD and system optimisation to simulate different aspects of product design and development, significantly reducing cost and time to market by allowing engineers to optimise the design before physical prototyping. CAE tools are grouped into thermomechanical, electromagnetic and fluid dynamics analysis, followed by free options for these applications.

    4.1.1 FEM Workflow

    Most of the listed CAE tools uses FEM (Finite Element Method), FDM (Finite Difference Method) or FVM (Finite Volume Method) for simulations, which mainly contains 3 stages.

    1. Pre-processing: Defining of the geometry, material properties, and boundary conditions of the system under study.
    2. Processing: Solving the mathematical equations using numerical methods.
    3. Post-processing: Analyzing and interpreting the results to draw conclusions.

    Most of the commonly used softwares have all these 3 stages embedded as a continuous process via a single window. This is much easier for the user. But this luxuary is not always available when it comes to free software. One might have to use upto 3 software for each mentioned stages to do an analysis. In some cases, this can be an advantages, especially many of these are modular - meaning one can use a suitable solver with a preffered pre or post processor. Also, automation & modification of the model during the simulation is also possible to an extent. https://any-answers.com/blog/what-is-pre-and-post-processing-in-fea/, https://www.ques10.com/p/18773/explain-pre-and-post-processing-in-fem-1/

    Pre-processing - 3D modeling, material definition, meshing, load/boundary conditions definitions.

    Processing - Solving the system using diagonal matrices.

    Post-processing - Visualization and analysis of the data.

    4.1.2 Proprietary Software

    Most of the listed software belong to the suite of tools provided by software giants such as Ansys, Altair, Dassault Systems..., thereby providing a smooth transition between sibling software. The previously mentioned 3 stages in the analysis (Pre-processing, processing & post-processing) are generally handled in a single window in most of these software.

    SolidWorks Flow Simulation Another CAD embedded CFD package, for simple systems. Limited with controls and fuctionalities. Better integration with SolidWorks (3D modeling).

    4.1.3 Free Software

    Unlike the proprietary software where all 3 stages n the analysis can be done in a single window, free software are mostly work as modules. There are various options to do the entire process inside a single window also, if one wishes, they can use multiple modules for various steps - 3D modeling, meshing, material & boundary definition, solving the PDEs, and visualization. This can be helpful to process the data in between the steps or for automation. All the listed software in table have pre-processor & processor & post processor integrated in using external mesh generators, solvers and other addons.

    Software

    Features
    FreeCAD
    • Fundamentally a CAD software, but a good platform to integrate various mesh generators & solvers (FEM Workbench).
    • Parametrical CAD, with external control using python.
    • For all applications - thermostructural, mechanical, CFD, electromagnetic & acoustics. Everything is possible as per the solver (Calculix, Z88, Elmer & OpenFOAM. Many solvers are being integrated now).
    • Easy integration of 3rd party packages.
    • Good documentation and community support.
    Salome Meca
    • Single software for all stages.
    • Multiphysics solver, Code_Aster is embedded
    • The solver is actively updated with novel models and techniques.

    • Focusing on thermomechanical analysis.
    Elmer
    • Multiphysics solver - thermomechanical, CFD, electromechanical analysis.

    • Thermomechanical and CFD analysis.

    4.2 CAD

    CAD (Computer-Aided Design) is used to generate the 2D & 3D models for of the design. Especially for the CAE analysis complex structures are generated usig CAD softwares. Unlike CAE software, CAD software are very similar in terms of funtionalities and user interface. Some of the common software and free alternatives are listed.

    Software

    Features
    Proprietary Software
    SolidWorks Easy to use, focusing on 3D modeling. Includes basic simulations which are helpful for quicker design modification.
    Autodesk Inventor Similar to SolidWorks considering the 3D modeling and simulations. It has better tools for certain areas such as gear systems.
    CREO / Pro-E Parametric modeling is possible, easier handling of assemblies and integration with ERP software (Enterprise Resource Planning).
    AutoCAD Most used CAD software even in 2023, commonly used for 2D drawings (Architecht)

    4.3 Numerical Analysis

    The term 'Numerical Analysis' does not encapsulate the deapth of what the listed software are capable of. MATLAB is such a tool where people in most of scientific field use it to perform mathematical operations, data analysis, simulation, not to include the vast collection of toolset for various application. Although it is difficult replace MATLAB and similar proprietary software, there are free alternatives to manage the process.

    Software

    Features
    Proprietary Software
    Matlab Ideal tool for researchers and engineers with a bit of programming. Has a vareity of optional toolsets, 3rd modules. Good community support.
    Wolfram Mathematica Similar to MATLAB, but doesn't have a huge following. Visualization tools and certain mathematical operations are better than MATLAB. Good for symbolic computation.
    Maplesoft Better for analytical solutions. Doesn't have the community support and packages like MATLAB.
    Free Software
    Gnu Octave Basically a free MATLAB clone, with slight variations in sytax and some functionalities. The libraries are not as big as MATLAB, but growing with the community support.
    Scilab Also similar to MATLAB, but not as close as Octave. Certain libraries are better than Octave, like Simulink alternative Xcos.
    Maxima Suitable for analytical operation over numerical analysis. Similar to Mathematica & Maplesoft.
    Python Widely used with huge community support and various packages for several applications. Unlike proprietary software, the required tools are in modulular form, like SciPy, NumPy & Matplotlib.
    Julia Similar to Python, but with improved operations and functionalities. Julia lacks the community support compared to Python. There are software and packages like Juno IDE (no longer supported, but integrated with Visual Studio).

  5. Workflow Example - Motor design with free software

    6. This is an example of how a complex system like an electric motor could be designed using free software. Various steps during the design phase is shown below. The listed process can go through multiple iterations before finalising the design.

    1. Deciding the motor type (Induction, PM ...) and general constraints (dimension, slot numbers...).
    2. Evaluating the output via electromagnetical analysis
    3. Thermal analysis
    4. Mechanical analysis - strength & vibrations
    5. Fluid dynamics

    For the steps mentioned above, there are various free softwares and online platforms. For the electric motor design FEMM is a nice tool to evaluate the electromagnetic properties.

    FEMM - Electromagnetical analysis for the design - slot number, winding... (Emetorand similar are useful).

    FreeCAD - 3D modeling of components and assemblies.

    FreeCAD - CAE analysis. Mainly thermomechanical analysis, if needed CFD as well.
